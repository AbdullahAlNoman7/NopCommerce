### Web Automation NopeCommerce

### Project Scenerio
1) Visit NopeCommerce
2) Registertion with valid data
3) Login with vaid user
4) Edit infomation
5) Add to cart product
6) Remove product from cart
7) Continue shopping
8) Logout
### Technology Used
- Tools: Selenium
- Language: Python
- Editor: PhCharm 
### How to use this project
- Clone this project 
- pip install -r requirements.txt
